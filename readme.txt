A Sidebar widget to display the co-Authors profile on a Single page (with co-Authors plugin)

== Description ==

This can display a sidebar widget on a single page to show the co-authors avatar and profile. 

Works with <a href="http://wordpress.org/extend/plugins/co-authors/">co-authors</a> plugin.

Inspired by <a href="http://wordpress.org/extend/plugins/author-profile/">Author Spotlight (Widget)</a> plugin.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `co-authors-spotlight_widget` directoy and copy it to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Choose the co-authors in the article edition (do not forget to show it in preferences).
